Если вы до этого момента не пользовались сборщиком GULP введите в
терминал команду npm i gulp-cli -g это установит GULP в систему глобально.

После завершения установки GULP введите команду npm i

------------------------------------------------------

1. Режим разработчика. Команда запуска npm run dev
	1.1 локальные шрифты npm run fonts 
2. Режим продакшена. Команда запуска npm run build




3. Режим продакшена и отправка результата на сервер по FTP.
Команда запуска npm run deploy


4. Режим продакшена и создание ZIP-архива с результатом. Команда
запуска npm run zip

5. Режим продакшена без создания WEBP изображений и действий
связанных с этим форматом. Команда запуска npm run devbuild
